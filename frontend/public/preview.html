This is a static preview of the ZTD frontend. The actual application includes:

  • Interactive task management (add, edit, delete, complete)
  • Real-time filtering by category and priority
  • Three view modes: Active, Completed, and By Category
  • Inline editing with save/cancel
  • Statistics dashboard with live updates
  • Fully responsive design for all devices
  • Beautiful animations and hover effects

To see the live application:

  1. Start the backend: cd backend && python app.py
  2. Start the frontend: cd frontend && npm start
  3. Open http://localhost:3000 in your browser
